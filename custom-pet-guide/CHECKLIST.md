# 반려동물 제작 체크리스트

## 입력 고정

- [ ] 사용할 기준 이미지 수와 파일명을 확정했다.
- [ ] 기준 이미지 SHA-256 목록을 기록했다.
- [ ] 얼굴·머리·상의·바지·신발의 정체성 계약을 작성했다.
- [ ] 후드, 지퍼 등 금지 요소를 명시했다.

## 아틀라스 품질

- [ ] 1536×2288, RGBA, 8열×11행의 v2 형식이다.
- [ ] 표준 0~8행의 보이는 높이 편차가 8% 이내다.
- [ ] 한 행 안에서 캐릭터 비율과 의상이 동일하다.
- [ ] 달리기 좌·우 방향이 화면 좌표 기준과 일치한다.
- [ ] 점프 행의 바지가 검은색으로 유지된다.
- [ ] 인접 프레임이 실제로 달라 정지 영상이 아니다.
- [ ] 잘림, 중복, 비정상 빈 칸이 없다.
- [ ] `r0c6` 중립 프레임이 있고 `r0c7`은 비어 있다.
- [ ] 투명 RGB 잔여값과 크로마키 테두리가 0이다.

## 설치와 검증

- [ ] 기존 설치본을 복구 가능한 위치에 백업했다.
- [ ] 공식 검증기와 보조 검증기를 모두 통과했다.
- [ ] 접촉 시트로 독립 시각 검수를 수행했다.
- [ ] 설치 위치에서 공식 검증을 다시 실행했다.
- [ ] 후보와 설치본의 SHA-256이 일치한다.
- [ ] 개인 이미지·절대 경로·설치 패키지를 커밋하지 않았다.
